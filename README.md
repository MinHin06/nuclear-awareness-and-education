# Nuclear Awareness & Education Website

## Overview

The Nuclear Awareness & Education Website is a responsive educational website developed to increase public awareness and understanding of nuclear energy, its applications, benefits, safety measures, and common misconceptions. The website provides users with accessible and informative content through a clean and user-friendly interface.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Informative pages about nuclear energy and radiation
- User-friendly navigation
- Modern and clean user interface
- Interactive elements for improved user experience

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
